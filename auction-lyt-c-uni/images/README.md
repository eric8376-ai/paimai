# 图片资源说明

## 占位图片

当前版本使用在线图片URL作为占位符:
- 默认卡片图片: `https://img.zgfczc.com/auction/image/auction-def.jpg`
- 页面背景图: `https://img.zgfczc.com/auction/image/page-bg.jpg`
- 使用指南图片: `use-guide.jpg` (需要本地文件)

## 建议操作

1. 从小程序源码复制图片资源到对应目录
2. 或使用占位图服务,如:
   - https://via.placeholder.com/
   - https://picsum.photos/

## 图片尺寸要求

- 使用指南横幅: 638 x 132 px (2:1)
- 卡片封面图:
  - 拍卖会/资产招商: 320 x 180 px (16:9)
  - 标的物: 自适应,建议 3:2 或 16:9
